Views are stored in the resources/views directory.

	

	
	    
	        
Hello,

Since this view is stored at resources/views/greeting.php, we may return it using the global view helper function like so:

Route::get('/', function ()    {
    return view('greeting', ['name' => 'James']);
});

Views may also be nested within sub-directories of the resources/views directory. "Dot" notation may be used to reference nested views. For example


return view('admin.profile', $data);
